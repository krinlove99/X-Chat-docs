## X-Chat

X-Chat是BASETEC旗下基于人工智能大语言模型的聊天助手应用。

## 声明

大模型应用发展速度超乎了所有人的预料，目前用户的关注度仍然集中于OpenAI及Anthropic，对于国产模型的关注度仍不够高。事实上，国产大模型的发展水平已经与第一梯队无差。

本项目积极推广和宣传国产大模型，致力于将人工智能发展的成果惠及每一位用户。
